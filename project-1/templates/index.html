About This Demo

This is a simple Flask application deployed to AWS Elastic Beanstalk. It demonstrates:

  • Basic Python application structure for Elastic Beanstalk
  • Configuration with .ebextensions
  • Simple API endpoints
  • Health checks

API Endpoints

  • /health - Health check endpoint
  • /api/info - Application information
